---
title: "Heap"
---

Add the following to your `mint.json` file to send analytics to Heap.

<CodeGroup>

```json Analytics options in mint.json
"analytics": {
    "heap": {
        "appId": "required"
    }
}
```

```json Example
"analytics": {
    "heap": {
        "publicApiKey": "1234567890"
    }
}
```

</CodeGroup>
